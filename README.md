# hla

The High Level Assembler (HLA) is a 32-bit 80x86 assembly language that is portable between Win32, Linux, and FreeBSD.

This project is a derivative work of High Level Assembler (HLA) by Randall Hyde,
which was released into the Public Domain.

- Original author: [Randall Hyde](https://www.randallhyde.com/)
- Original project: [sourceforge.net/projects/hlav1](https://sourceforge.net/projects/hlav1/)

## License

The original repository created by [Randall Hyde](https://sourceforge.net/u/rhyde/profile/) states "Public Domain" as lincese.

Since modifications might happen the MIT license has been selected in order to keep the original author contribution separated by any possible further modification. The choice is based on the principle the original author would not be forgotten but also not involved with 3rd-party changes. Under this perspective, the MIT license seems a good enough candidate and BSD-3 clause license which has an explicit "no endorsement" clause the other one suitable. The MIT license is a bit more widely known, thus accepted as a legal standard, and it includes explicitly the AS-IS clause which is aligned with the prototype maturity level claimed by the original author.
